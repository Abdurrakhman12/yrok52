Главная Описание Отзывы

Описание

Основной задачей игрока в данной версии игры является создание своей команды с высоким общим рейтингом, как и в основной игре в режиме Ultimate Team[1][2]. В FIFA Mobile 22 появилась возможность условно бесплатно создать свою форму в режиме Ultimate Team. В мобильной версии присутствуют все клубы, которые есть в основной версии симулятора — FIFA 22. Режим карьеры в мобильной версии отсутствует, однако есть множество других режимов: играть можно либо с ботами, либо с реальными геймерами.

Отзыв

На данный момент на мой взгляд эта игра является лучшей спортивной игрой для телефонов и планшетов. В игре несколько видов управления, каждый сможет выбрать для себя, кому уж как удобнее. Сам игровой процесс и графика...

Абдуррахман Кайсагали
